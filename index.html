Key Features

Active Directory Integration

Centralized user and group management with seamless Active Directory integration for enterprise authentication and authorization.

Automated Drive Mapping

Automated drive mapping for each department via GPO with secure file storage on TrueNAS SMB shares with quota management.

Cloud Backup

Secure cloud backup on AWS S3 with TrueNAS sync for disaster recovery and business continuity with versioning capabilities.

Monitoring & Alerting

Comprehensive monitoring and alerting with Prometheus and Grafana integration for system health and performance tracking.

Admin & User Portal

Web and mobile portal for admin and users with role-based access control and self-service features for file management.

Secure Remote Access

Secure remote access with RDP capabilities for administrators and role-based access controls with multi-factor authentication.

System Architecture

Integrated Enterprise Solution

pe-connect is a comprehensive enterprise file management system that plays a crucial role in modern organizational infrastructure by providing:

  • Centralized user management through Active Directory integration
  • Automated department-based file storage with Group Policy Objects
  • Secure, scalable file storage using TrueNAS SMB shares
  • Robust disaster recovery with AWS S3 cloud backup
  • Intuitive web portal for both administrators and end-users
  • Comprehensive system monitoring with Prometheus and Grafana

How pe-connect Works

1

User Authentication

Users authenticate through Active Directory, which provides centralized identity management and single sign-on capabilities across the organization.

2

Drive Mapping

Group Policy Objects automatically map network drives based on department membership, ensuring users have immediate access to appropriate resources.

3

File Storage

TrueNAS provides reliable SMB shares with quota management, access controls, and high availability for departmental file storage.

4

Cloud Backup

Automatic synchronization with AWS S3 ensures all critical data is backed up to the cloud with versioning for disaster recovery purposes.

5

Monitoring

Prometheus collects metrics from all system components, while Grafana provides visualization and alerting for proactive system management.

6

Web Portal

The responsive web interface allows administrators to manage users and files, while providing self-service capabilities to end-users.
